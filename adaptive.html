Adaptive algorithms¶

Warning

These adaptive algorithms are (currently) experimental and may change at any time. Do not use these adaptive algorithms in deployment.

There are many queries to ask about in triplet queries: with \(n\) objects there are \(\mathcal{O}(n^3)\) questions that could be presented.

So what question should be asked? That’s the task of an adaptive algorithm. The API the must conform to below:

salmon.backend.alg.Runner(name)

Run an adaptive algorithm.
