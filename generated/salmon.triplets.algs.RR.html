A randomized round robin algorithm.

Parameters
dint

Embedding dimension.

R: int = 1

Adaptive sampling starts are R * n response have been received.

optimizerstr

The optimizer underlying the embedding. This method specifies how to change the batch size. Choices are ["Embedding", "PadaDampG", "GeoDamp"].

optimizer__lrfloat

Which learning rate to use with the optimizer. The learning rate must be positive.

optimizer__momentumfloat

The momentum to use with the optimizer.

scorerstr, (default "infogain")

The scoring method to use.

modulestr, optional (default "TSTE").

The noise model to use.

kwargsdict

Arguments to pass to ~Adaptive.
